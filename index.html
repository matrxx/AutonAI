How to use the Autonomous Multi-Agent System:

  • Start a project: Type "start project: [your project description]" to begin
  • Get status updates: Click the "Get Status Update" button or ask "What's the current status?"
  • Talk to specific agents: Address them by name (e.g., "FrontendDev, create a responsive navbar")
  • Upload documents: Provide additional context or requirements for your project
  • View console: Click "Show Console" to see detailed system logs
  • Reset the system: Click the "Reset System" button to clear agent memories and start fresh

Example: start project: Create a landing page for an Italian restaurant with information about their menu, location, and a reservation form
